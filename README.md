# Student-Management-Java-Html-project-
A beginner-friendly project built in both Java and HTML/JavaScript to manage student records. It supports adding, searching, displaying, and deleting students. The Java version runs in the console, while the web version works in the browser, helping understand both backend logic and frontend basics.
# Student Management System

## Overview

This is a beginner-friendly **Student Management System** implemented in two versions:

* **Java (Console-based)** for understanding core backend logic and OOP concepts.
* **HTML + JavaScript (Web-based)** for learning frontend basics and dynamic UI handling.

The project helps beginners practice real-world CRUD operations in a simple and understandable way.

---

## Features

* Add student details (Name, ID, Age, Department)
* View all students
* Search student by ID
* Delete student by ID
* Input fields auto-clear after actions

---

## Tech Stack

### Java Version

* Java (Core Java)
* OOP concepts (Classes, Objects)
* ArrayList

### Web Version

* HTML

## How to Run

### Java Version

1. Place `Main.java`, `Student.java`, and `StudentService.java` in the same folder
2. Open terminal in that folder
3. Compile:

   ```bash
   javac *.java
   ```
4. Run:

   ```bash
   java Main
   ```

### HTML Version
